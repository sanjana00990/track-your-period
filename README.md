# Track Your Period

A simple and intuitive web-based period tracking application that helps users monitor their menstrual cycle, predict upcoming periods, and stay informed about their health.

## Live Demo
https://track-your-period.vercel.app/

## Features

* Track period start and end dates
* Predict next cycle based on previous entries
* Clean and user-friendly interface
* Lightweight and fast (runs in browser)
* No backend required


## Tech Stack

* HTML5
* CSS3
* JavaScript


## How to Run Locally

1. Clone the repository
git clone https://github.com/sanjana00990/track-your-period.git

2. Navigate to the project folder
cd track-your-period


3. Open the app

* Double-click `index.html`
  OR
* Open with Live Server in VS Code
